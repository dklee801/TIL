# 파이썬 가상환경



## 사용법



#### - 가상환경 생성

- venv로 가성환경을 생성



- 가상환경을 생성하면 해당 디렉토리에 venv폴더 생성

```bash
$ source venv/Scrpt/activate

$ source venv/Scrpt/activate

$ source venv/Scrpt/activate

```

- 가상환경을 실행시킨 상태에서 파이썬 패키지(pip)를 설치하게 되면 venv 폴더내의 lib폴더에 설치하게 됨

## pip

```bash
#requirements.txt 에 설치된 패키지 기록
$pip free > requirement

require -r requirement

```
